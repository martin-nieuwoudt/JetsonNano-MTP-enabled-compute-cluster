NVIDIA has heavily integrated the Model Context Protocol (MCP) across its developer ecosystem, enabling AI agents and LLMs to seamlessly access complex NVIDIA domain tools. You can build, deploy, or connect to these MCP capabilities using primary developer resources:
NVIDIA NeMo Agent Toolkit: Functions as both an MCP client and server, allowing you to connect remote tools or publish customized agent functions.
NVIDIA AIQ Blueprint: Uses MCP as a standardized framework to discover and connect external data sources or agent apps to your workflows.
NVIDIA RAG MCP Server: Acts as an adapter to instantly connect your RAG and ingestion systems to MCP-aware clients like IDEs and agent orchestration frameworks.
NVIDIA RTX Remix MCP: Automatically initiates an MCP server via Server-Sent Events (SSE) so that AI agents can learn about and interact with your Remix environment. [7]
